ボレンプロジェクト
====

Overview
Vollenhovia属に分類されるヤドリウメマツアリとウメマツアリの生体情報をもとに、両種における巣仲間認識の違いを探索する。

## Description
ウメマツアリには遺伝子情報を付与されたゲノムデータが公共データベースに登録されているが、ヤドリウメマツアリはトランスクリプトームデータのみしか登録されてない。
そこでトランスクリプトームデータをもとに擬似的なゲノムデータを作成し、両種の配列類似性解析を行う。

## Requirement
- python3.7
- Briger
